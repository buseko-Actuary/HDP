# Heart Disease Predictor

A binary classification model that predicts whether a patient is likely to have heart disease from clinical measurements — age, sex, chest pain type, cholesterol, resting blood pressure, and other cardiac indicators.

## The problem

Heart disease is one of the leading causes of death worldwide, making early detection critical. This project applies data preprocessing, exploratory data analysis, and machine learning classification to identify patterns and risk factors associated with heart disease, aiming to support earlier, more accurate diagnosis.

## What this covers

- Cleaning and exploring a clinical dataset (age, sex, chest pain type, resting blood pressure, cholesterol, fasting blood sugar, ECG results, max heart rate, exercise-induced angina, and more)
- Building a classification model to predict a binary heart disease outcome
- Evaluating the model's predictive performance
- Identifying which clinical features are most strongly associated with heart disease risk

## Stack

Python · pandas · numpy · seaborn/matplotlib · scikit-learn (classification models)
