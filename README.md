## 🎯 The problem

Heart disease is one of the leading causes of death worldwide, making early detection critical. This project applies data preprocessing, EDA, and machine-learning classification to identify **patterns and risk factors** associated with heart disease — aiming to support earlier, more accurate diagnosis.

## 🔍 What it does

| Step | Detail |
|---|---|
| 🧹 Clean & explore | Clinical dataset: age, sex, chest-pain type, resting BP, cholesterol, fasting blood sugar, ECG, max heart rate, exercise-induced angina, and more |
| 🤖 Model | Build a classifier for the binary heart-disease outcome |
| 📈 Evaluate | Measure predictive performance |
| 🧠 Interpret | Surface the clinical features most strongly linked to risk |

## 🛠️ Stack
